假定把一个二维数组的元素从最后一行开始， 每一行从右至左进行映射

1）按这种映射次序列出数组score[3][5]的索引
map(i1, i2) = -5*i1 - i2 + 14

注：可以通过解线性方程组解决
假设map(i1, i2) = m*i1 + n*i2 + c
已知map(2, 4) = 0, map(2, 3) = 1, map(2, 2) = 2

2）给出数组score[u1][u2]的映射函数
若u2 = 1 && u1 = 1:
map(i1, i2) = 0

若u1 = 1 && u2 != 1:
map(i1, i2) = u2 - 1 - i2

若u2 = 1, u1 != 1
map(i1, i2) = -i1 + u1 - 1

若u1 != 1 && u2 != 1:
map(i1, i2) = -u2*i1 + [(1-u2)/(u2-1)]*i2 + u1*u2 - 1
